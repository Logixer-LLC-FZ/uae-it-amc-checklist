# UAE IT AMC Checklist

A practical IT AMC and preventive maintenance checklist for businesses in Dubai and across the UAE.

This checklist is created by **Logixer LLC FZ** to help small and medium businesses understand the key areas that should be reviewed regularly to reduce downtime, improve security, and maintain reliable IT operations.

## What is IT AMC?

IT AMC stands for **IT Annual Maintenance Contract**. It is a support agreement that helps businesses maintain computers, servers, networks, cybersecurity systems, VoIP phones, backups, and other IT infrastructure through regular preventive maintenance and technical support.

Instead of waiting for problems to happen, an IT AMC helps businesses detect issues early and keep systems running smoothly.

## Monthly IT AMC Checklist

### 1. Computer and Laptop Maintenance

* Check system performance
* Remove unnecessary temporary files
* Review storage usage
* Check operating system updates
* Verify antivirus status
* Check user login issues
* Review hardware health

### 2. Server Maintenance

* Check server uptime
* Review CPU, RAM, and disk usage
* Verify event logs
* Check user access permissions
* Confirm backup completion
* Review critical alerts
* Check storage capacity

### 3. Network and Wi-Fi Checks

* Test internet connectivity
* Check firewall status
* Review router and switch performance
* Test office Wi-Fi coverage
* Check network speed
* Verify VLAN or guest network settings
* Review connected devices

### 4. Cybersecurity Checks

* Confirm antivirus and endpoint protection status
* Review firewall rules
* Check suspicious login attempts
* Ensure multi-factor authentication is enabled where possible
* Review email security settings
* Check password policy
* Educate staff about phishing risks

### 5. Backup and Data Protection

* Confirm backup schedule
* Test backup restore process
* Check cloud backup status
* Review backup storage capacity
* Verify important business files are included
* Check ransomware protection settings

### 6. VoIP and PBX Maintenance

* Test incoming and outgoing calls
* Check SIP trunk status
* Review call quality
* Verify extension settings
* Check call recording storage
* Review IVR and call routing
* Test remote extensions if applicable

### 7. CCTV and Access Control

* Check camera recording status
* Review storage capacity
* Test remote viewing access
* Check access control logs
* Verify biometric or card access devices
* Review user permissions

## Why Preventive IT Maintenance Matters

Regular IT maintenance helps businesses:

* Reduce downtime
* Improve employee productivity
* Protect business data
* Avoid unexpected repair costs
* Improve cybersecurity readiness
* Extend the life of IT equipment
* Keep communication systems reliable
